# Baby Care Market Analysis
📊 Data Analysis Portfolio 🎯 DATASET: Real market research data (Baby_Care_sourceDA.xlsx) 


## 📌Project Overview

This Project analyzes consumer trends in baby care market using Excel,SQL and Power BI.  
The goal was to uncover insights into purchasing behavior and highlight growth opportunities.

## 🛠️ Tools & Technologies
- SQL (data cleaning & transformation)
- Power BI (dashboard creation)
- Excel (initial dataset handling)

## 📊 Key Insights
- Built dashboards reveal trends by category and consumer segment.

## 🚀 Impact
- Improved decision‑making with clear, visualized insights.

- `data/` → Sample datasets (cleaned & raw)
- `scripts/` → SQL queries for cleaning and analysis
- `dashboards/` → Power BI files and screenshots
- `README.md` → Project documentation
